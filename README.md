# Budget Calendar Tracker

A monthly budget tracking web app with a calendar view. Track daily expenses, manage subscriptions, and monitor your spending at a glance.

## Features

- **Calendar View** - See daily spending on an interactive monthly calendar
- **Daily Expenses** - Add, view, and delete expenses for any day
- **Subscriptions** - Track recurring monthly costs with due dates
- **Budget Summary** - Cards showing total spent, remaining balance, and daily average
- **Monthly Summary** - Expandable table comparing budget vs. spending across all months
- **Light/Dark Theme** - Toggle between themes with preference saved locally
- **Cross-Device Sync** - Data syncs via Supabase with email or Google sign-in

## Tech Stack

- **TypeScript** + **Vite**
- **Supabase** (auth & data sync)
- Vanilla DOM (no framework)

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build

```bash
npm run build
npm run preview
```
